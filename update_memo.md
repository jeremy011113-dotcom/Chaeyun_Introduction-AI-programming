# 업데이트 메모 (robot_game.py)

**변경 사항:**
- 게임 실행 중 언제든지 사용자가 게임을 중단할 수 있는 기능(중도 포기)을 추가했습니다.
- 입력창(팀 선택, 공격 대상 선택, 행동 선택 등)에 `[종료: q]` 안내가 표시됩니다.
- 입력을 받을 때 `q` 나 `quit` 을 입력하면 게임 프로세스가 즉시 안전하게 종료(`sys.exit(0)`)되도록 구현했습니다.

**개발 세부 내용:**
- `Game` 클래스 내에 사용자 입력을 공통으로 처리하는 `get_input()` 메서드를 새롭게 작성하여 중복을 줄였습니다.
